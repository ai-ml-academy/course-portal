# AI/ML Academy — Course Portal

Public-facing course materials for the Gen AI Engineer program.

## Structure

```
course-portal/
├── index.html              ← Landing page
├── quizzes/
│   └── class-NN/index.html ← Per-class quizzes
└── slides/
    └── class-NN/index.html ← Per-class slide decks
```

## Deployment

Hosted via GitHub Pages from the `main` branch.

**Live at:** `https://ai-ml-academy.github.io/course-portal/`

## Adding Content

- **Quiz:** Copy HTML to `quizzes/class-NN/index.html`, add link to `index.html`
- **Slides:** Copy HTML to `slides/class-NN/index.html`, add link to `index.html`
